我基于PyQt5设计了用于控制相机和投影仪的界面，其中相关控件的功能如下：
Open 用于打开相机和投影仪，关联到打开相机的函数和打开投影仪并投影全白条纹的函数；
Left_show 用于展示左侧相机流取到的图像，鼠标置于label内部之后可以通过滑轮去放大或者缩小图像；
Right_show 用于展示右侧相机流取到的图像，鼠标置于label内部之后可以通过滑轮去放大或者缩小图像；
Disconnet 用于关闭相机流取和投影仪投影仪，并断开和相机投影仪的连接；
change_exposure 用于修改相机的曝光参数（EXPOSURE_TIME_US），修改完之后Left_show和Right_show实时改变流取的参数；
Crosshair 用于分别在 Left_show和Right_show 控件中添加一个十字线；
choose_pro 选择第一个投影全白画面，选择第一个投影棋盘格，选择第三个投影仪中心线；
Change_brightness 用于修改投影仪的亮度（PROJECTOR_BRIGHTNESS），修改完之后投影仪硬件实时调整的亮度；
Warning 用于提醒大家是否打开设备，如果相机和投影仪都成功打开控件变绿，如果Scan_rebuild开始扫描控件变红；
Change_stripe 选择第一个的时候投影的第一张图像的起始序号（START_FRAME: int = 0），投影图像数BURNED_PATTERN_COUNT = 18，选择第二个的时候投影的第一张图像的起始序号（START_FRAME: int = 18），投影图像数BURNED_PATTERN_COUNT = 36，选择第三个的时候投影的第一张图像的起始序号（START_FRAME: int = 54），投影图像数BURNED_PATTERN_COUNT = 8，选择第四个的时候投影的第一张图像的起始序号（START_FRAME: int = 62），投影图像数BURNED_PATTERN_COUNT = 9；
Scan_rebuild 用于调用投影仪和相机联合采集条纹图；
photo 用于触发左右相机单帧拍照并保存在captures文件夹中；
open_folder 用于打开Scan_rebuild左右相机采集到的图像的父文件夹。
